# 機能要件

## 機能一覧

| 機能カテゴリ | 機能 | 説明 |
|------------|------|------|
| 認証 | ログイン（未実装） | 登録済みのID・パスワードで認証 |
| 認証 | ログアウト（未実装） | セッションを破棄してログイン画面へ遷移 |
| カード | 作成 | タイトル・説明文・優先度・期限を入力してカードを追加（タイトル・期限は必須） |
| カード | 編集 | 既存カードのすべての項目を編集 |
| カード | 削除 | 確認ダイアログを表示してカードを削除 |
| カード | 移動 | ドラッグ＆ドロップでカラム間を移動 |
| カラム | 固定表示 | Todo / In Progress / Done の3列を表示 |
| カラム | 並び替え | 各カラムのカードを優先度・期限でワンショット並び替え |
| 検索 | キーワード検索 | タイトルをキーワードで絞り込み（部分一致・大文字小文字無視） |
| 検索 | ステータスフィルター | 指定したステータスのカードのみ表示 |
| データ保存 | 永続化 | バックエンド経由でデータベースに保存 |

## 機能詳細

### 認証

> ※ 未実装。将来の実装予定。

- ユーザーは1名のみ（シングルユーザー）
- 未認証状態でボード画面へアクセスした場合、ログイン画面へリダイレクト

### カード（タスク）

| 機能 | 説明 |
|------|------|
| 作成 | タイトル・説明文・優先度・期限を入力してカードを追加（タイトル・期限は必須） |
| 編集 | 既存カードのすべての項目を編集 |
| 削除 | 確認ダイアログを表示してカードを削除 |
| 移動 | ドラッグ＆ドロップでカラム間・カラム内を移動。移動後の順番はDBに保存され、再読み込み後も維持される |

### カラム

| 機能 | 説明 |
|------|------|
| 固定表示 | Todo / In Progress / Done の3列を表示 |
| 並び替え | カラムヘッダーのボタンで優先度順・期限順にワンショット並び替え（DBには保存しない）。並び替え後の表示順はブラウザ（localStorage）に保存され、再読み込み後も維持される。ドラッグ＆ドロップによる手動並び替えも可能で、その順番はDBに保存される |

### 検索・フィルタリング

| 機能 | 説明 |
|------|------|
| キーワード検索 | 入力したキーワードでタイトルを部分一致検索する（大文字小文字は区別しない） |
| ステータスフィルター | 選択したステータス（Todo / In Progress / Done / すべて）でカードを絞り込む |

### データ保存

- ブラウザを閉じてもデータが消えない
- バックエンド（Java）経由でデータベースに保存
